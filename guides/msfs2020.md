# Microsoft Flight Simulator 2020

## install 
- install via steam
- install protonup => https://aur.archlinux.org/packages/protonup-qt-bin
- install ge-proton
- change proton for msfs to ge-proton
- dereferance wine prefix
```
Make a backup copy of the ../<gameid>/pfx folder     	#That is the wineprefix for the proton environment
rm -r ../<gameid>pfx/dosdevices 				#This is all symolic links that we don't want to dereference
cp -RLv ../<gameid> ../<gameid>.dereferenced		#Makes a copy with all symlinks replaced by their target files
copy the ../dosdevices directory from ../<gameid>.backup to ../<gameid>.derefereced
rm -r ../<gameid>						#Remove the original
Copy ../<gameid>.derefereced ../<gameid>			#Now the wineprefix only contains files, no symlinks
```
- install win dependencies
```
protontricks 1250410 win7
protontricks 1250410 -q remove_mono
protontricks 1250410 -q dotnet48
```

## install vPilot and YourControl
```
protontricks 1250410 --gui
```
- launch uninstaller
- click install

## launch exe independendly
```
protontricks-launch --appid 1250410 $HOME/.local/share/Steam/steamapps/compatdata/1250410/pfx/drive_c/...
```
